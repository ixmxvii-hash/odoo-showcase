# City/Industry Pages Plan

## Overview
24 pages total (3 cities x 8 industries) with:
- **Header**: Odoo implementation benefits contextualized to each city's market
- **Body**: Detailed breakdowns with charts/graphs showing verified Odoo results

---

## Page Structure

### 1. Hero Section (Header)
**3 Stats** showing Odoo benefits in context of local market:

```
[Stat 1: Efficiency/Productivity]  [Stat 2: Time/Cost Savings]  [Stat 3: ROI/Implementation]
```

### 2. Body Sections

#### A. "Why [City] [Industry] Companies Choose Odoo"
- Bar chart showing before/after metrics
- 3-4 bullet points with verified improvements

#### B. "Results [City] Businesses Achieve"
- Donut/pie chart showing efficiency breakdown
- Timeline graphic showing implementation phases

#### C. "Local Success Metrics"
- Line chart showing ROI over time
- Comparison table vs legacy ERP systems

## Data Structure Changes

```typescript
interface CityIndustryOverride {
  // Hero Section
  heroTitle: string;
  heroSubtitle: string;
  heroDescription: string;
  stats: {
    value: string;
    label: string;
    context: string;  // City-specific context
    source?: string;  // Verification source
  }[];

  // Body Section - NEW
  bodyStats: {
    beforeAfter: { metric: string; before: string; after: string; improvement: string }[];
    roiTimeline: { month: number; value: string; label: string }[];
    implementationWeeks: number;
    costComparison: { legacy: number; odoo: number };
  };

  // Existing
  localZones: string[];
  zipCodes: string[];
  testimonials: Testimonial[];
}
```
